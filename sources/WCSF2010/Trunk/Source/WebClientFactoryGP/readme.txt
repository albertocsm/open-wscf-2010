Modifying the Web Client Factory Guidance Package

To customize and compile the guidance package, the following need to be installed:

1) Install: Guidance Automation Extensions 2010

2) Install: Guidance Automation Toolkit 2010

3) copy AjaxControlToolkit.dll(version 3.0.11119) to \BlocksTrunk\Source\Lib\AjaxControlToolkit3.5\ 

to downolad AjaxControlToolkit-NoSource.zip:

**browse to : http://ajaxcontroltoolkit.codeplex.com/releases/view/8513

**click on: AjaxControlToolkit-Framework3.5-NoSource.zip and save the zip file

   Note:

   don't click on AjaxControlToolkit-NoSource.zip. That zip contains 1.0 version of the DLL, which is not what you want.

**copy AjaxControlToolkit-Framework3.5-NoSource.zip\SampleWebSite\Bin\AjaxControlToolkit.dll to this folder.

   Note: Please check the property of the DLL to make sure that the version number is 3.0.11119.
